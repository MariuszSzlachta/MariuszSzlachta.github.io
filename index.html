About me

My name is Mariusz. I'm an ambitious web developer with a creative approach to solving problems and huge dose of sense of humor. Creating applications and websites gives me great satisfaction. I love that feeling when complex parts of code become clear and problems, lately imposible to solve, becomes easy to overwcome.

I'm a graduate of the Web Developer bootcamp organized by Kodilla, during which I built solid foundation in the area of web apps development based on the most popular Javascript technologies. In my projects I'm eager to use solutions offered by ReactJS and state management provided by Redux. SASS in SCSS syntax became my friend, and task automation using npm and building applications using a webpack is almost an indispensable standard in my work. It's g­reat that during the bootcamp I was able to get familiar with backend technologies as MongoDB with Mongoose, Node.js, Express. However, the most valuable thing I've learned is that nothing is impossible.

I'm currently looking for a job in which I will be involved in ambitious and interesting projects. I want to constantly develop my skills and learn from more experienced team members. I invite you to familiarize yourself with the most interesting projects I have carried out.

Project details

Avax

Landing page | August 2018

This cool landing page I created from *.psd file. This was very instructive project for me.

Many sections of this project pushed me to find best solition. Instead of using regular css I have decided to use scss, and its advantages like splitting components and section to multiple files, variables, and nestings. Avax uses custom theme google maps setted using its api. This approach allows to enjoy maximum performance and look control. Lighweight free carousel Flickity is one of my favorite. Despite its small weight provides many useful features and can be controlled using vanilla js what was my goal, because I didn't want to use JQuery. I need to mention about my script for filtering projects. I'm very proud of it because I could combine 'data-set' and javascript to make it works.

It's not possible to write everything just in several sentence. So then I encourage you to get familiar with source code and see how it works online, by clicking buttons down bellow.

Project details

Sudoku

Game | July 2018

Here is my version of this common game.

This project was created during Web Developer bootcamp. To do my game I used free library Sudoku-umd that generates sudoku. To build front of my app I decided to use React and its 'reactive' rendering was great solution, because I wanted to rerender only that field that changes. I decided to use Redux for state management, that made my state more predictable and it was great opportunity for practicing Redux with React. I tried to create a fully responsive game with functionalities that will allow you to choose the level of difficulty, show hints and even solve the game for you. I hope you will be enjoyed.

You can try yourself in sudoku by clicking 'go live' or watch my code and read readme file by clicking 'view code'.
